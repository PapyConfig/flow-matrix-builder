# Flow Matrix Builder\n\nStatic web tool for building network flow matrices. Define objects, VIPs, firewall rules → export CSV. Modular architecture for vendor-specific config generation (Fortigate, etc.).\n\n## Architecture\n\n- **Core**: data model + state management (vanilla JS modules)\n- **Forms**: UI components for each entity type\n- **Exporters**: pluggable output generators (CSV, future: Fortigate CLI)\n- **No backend** — fully client-side\n
